# gulp构建的前端项目
## 一、目录介绍
- config    gulp配置项
- data      测试的json数据
- dev       开发编译后的目录
- build    生成环境编译后的目录
- tpl       模板目录，每个模板以.tpl结尾
- src       源代码
  + css      css文件
  + js
  + img
  + lib      公共的组件
- .eslintrc.js   eslint规则
- .babelrc   ES6规则
- gulpfile.js
- package.json


## 二、功能列表
- 压缩代码
- less
- 实时编译
- 自动刷新
- ES6
- server
- 代码校验
- px转rem
- 添加css3前缀
- 模板编译



## 三、命令
### 初始化 cnpm install
### 开发： gulp run
### 部署： gulp build
